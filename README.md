## ðŸ›  Development

1. Clone the repo:
   ```bash
   git clone https://github.com/bepost/MorningMirror.git
   cd MorningMirror
   ```

2. Install dependencies:
   ```bash
   npm install
   ```
   
3. Build:
   ```bash
   npm run build
   ```
   
4. Load the extension in Chrome:

   * Go to `chrome://extensions/`
   * Enable **Developer mode**
   * Click **Load unpacked**
   * Select the `dist` folder
